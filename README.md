# imageRy_paper

Code and supporting material for the paper on the **imageRy** R package.

This repository collects the script used to illustrate the analytical workflow presented in the paper on **imageRy**, an R package designed for teaching and learning ecological remote sensing with accessible, reproducible tools.

## Repository structure

```text
imageRy_paper/
├── Visualizaion_plot.R
├── additional_scripts/
├── README.md
````

### Files and folders

* **`Visualization_plot.R`**
  Main script containing the workflow used to build the Figure of the manuscript.

* **`Additional_scripts/`**
  Additional scripts.

* **`README.md`**
  Project overview and guide to the repository contents.
